# byteskode-android-push

byteskode push - Android FCM library

## Installation
Add it to your build.gradle with:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
and:

```gradle
dependencies {
    compile 'com.github.byteskode:byteskode-android-push:{latest version}'
}
```

## Test
```sh
./gradlew test
```
